# Build a Formik Form

A simple example of using the Formik package to build a form with initial values, getting values in and out of the form state, validation and error messages, and handling form submission.

## Installation

1. Simply clone the repository
2. Run the script "npm start"
3. Open your web browser to "http://localhost:3000/"

## Usage

Once the webapge is loaded you should see the form with which you can play around with.
